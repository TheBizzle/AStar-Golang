AStar-Golang
===============

## What is it?

An A* implementation in Golang, building upon [my PathFindingCore module](https://github.com/TheBizzle/PathFindingCore-Golang).  It implements unidirectional A* with three different heuristics.

## Why is it?

I just do this when learning new programming languages.

See [here](https://github.com/TheBizzle/PathFinding) and [here](https://github.com/TheBizzle/AStar-Haskell) for similar examples.
